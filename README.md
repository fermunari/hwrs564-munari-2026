
Project Title: 
# Using Water Stable Isotopes and Tritium to Characterize the Age and Travel Paths of the Santa Cruz Headwaters

### Simple overview of use/purpose:
This project uses water stable isotopes (δ¹⁸O, δ²H) and tritium (³H) to investigate groundwater recharge sources, age, and flow paths in the headwaters of the Santa Cruz River (San Rafael Valley, Arizona).

### Description:

This project focuses on understanding groundwater systems in a semi-arid headwater basin using hydrogeochemical and isotopic tools. By analyzing stable isotopes and tritium in groundwater, springs, and surface water, this study identifies recharge sources (e.g., summer monsoon vs. winter precipitation), estimates groundwater residence times, and evaluates connections between groundwater and surface water systems.

The work is motivated by the need to better understand water availability and sustainability in the San Rafael Valley, an ecologically important and relatively understudied region. Results from this project contribute to improving conceptual hydrogeologic models and informing water resource management in arid environments.

### About this repository: 

The repository includes scripts and notebooks to generate:

δ²H vs δ¹⁸O plots for recharge interpretation
Piper diagrams for water type classification
Tritium plots for groundwater age estimation
